Board Game — Classical Notes

Designing a board game that teaches Indian classical dance & music through playful mechanics and tactile tokens — a blend of education and play.

Role: Game design, illustration, packaging.
Tools: Illustrator, Procreate, InDesign.

Project Overview

Rasa Quest is a board game designed to introduce children (6–12 years) to Indian classical music and dance through play. The game simplifies concepts like ragas, rasas, instruments, and classical dance forms using visual cards, symbols, and a strategy-based grid board. Inspired by texts like the Nāṭyaśāstra and Saṅgīta Ratnākara, it blends emotion, rhythm, and storytelling into interactive learning. The project demonstrates how communication design can make cultural heritage accessible, engaging, and fun for young learners.

About the Game

This game introduces children to Indian classical arts through guessing and strategy. Players work in teams to decode clues about classical instruments, dances, rasas, and concepts. Every correct guess lets them place a token on a 10×10 grid. The aim is to connect five tokens in a row, using strategy to build paths or block opponents. Simple questions, cultural clues, and tactical placement make the game both educational and exciting—turning classical arts into a playful, accessible experience for young learners.

Personal Takeaways

Working on this project helped me reconnect deeply with the classical arts I grew up around. Translating complex traditions into a playful learning tool taught me how design can simplify, preserve, and celebrate culture. I learned to balance research with creativity, and this process strengthened my confidence in using design to make heritage meaningful for young audiences.
